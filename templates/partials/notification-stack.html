{# Notification stack — one container above the navbar that holds the project's three persistent-ish notifications as a deck of cards: front → urgent / actionable (Update available, after a SW update) mid → soft CTA (Install this app) back → contextual (Translation in progress) The translation card is server-rendered on non-English pages. The install + update cards are present in the DOM at all times but hidden until the JS controller (notification-stack.js) reveals them — install on `beforeinstallprompt`, update on SW state change. Dismissals are written to localStorage by the controller: - translation: woh-banner-dismissed.translation (forever) - install: woh-banner-dismissed.install (until next prompt) - update: woh-banner-dismissed.update. (until next SW rev) See sass/components/_notification-stack.scss for the visual rules. #}
