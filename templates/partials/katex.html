{# Self-hosted KaTeX math rendering (CSP allows only same-origin assets, so no CDN). Included only on pages whose rendered content actually contains a `$$` display-math block — see the gate in base.html — so the ~0.5 MB of JS + fonts never loads on pages without math. Delimiters are deliberately limited to `$$…$$`, `\[…\]` and `\(…\)`. Bare single-`$` inline math is NOT enabled, so a literal dollar sign in prose can't be mistaken for the start of an equation. #}
