{# Scripture quote shortcode. Pulls one verse, or a verse range, from a hosted book in /library/ and renders it in a styled block that mirrors the library reader's interlinear view: the canonical translation above, the original-language source beneath, plus a "Read in " deep link to the verse anchor. Inline form (no body) — the canonical translation and original are loaded from the library JSON, so the timeline source never duplicates the verse text: {% scripture(book="genesis-woh", chapter=1, verse=6) %} {% scripture(book="genesis-woh", chapter=1, verse=6, to_verse=8) %} Body form (paired) — overrides the displayed translation with the author's own rendering (transliteration, a more literal gloss, an intentional ellipsis…) while still showing the canonical original line and the library link. {% scripture(book="genesis-woh", chapter=1, verse=6) %} And Elohim said: let there be a firmament in the midst of the waters. {% end %} Hidden behaviour: - The book title and original-language code are looked up from `data/library//_meta.json`. Missing meta = quiet fallback to the slug as the title and `und` as the language. - If `data/library//chapter-.json` is missing, only the body text and the link are rendered — useful for traditions we cite but don't (yet) host. - Verse numbers track the JSON's `paragraph.n` field, which is the same identifier the library reader uses for its `#cp` anchors, so deep links and shortcode citations stay aligned. #} {%- set _book = book | trim -%} {%- set _chapter = chapter -%} {%- set _verse = verse -%} {%- set _to_verse = to_verse | default(value=_verse) -%} {%- set _body = body | default(value="") | trim -%} {%- set _meta = load_data(path="data/library/" ~ _book ~ "/_meta.json", required=false) -%} {%- set _book_title = _book -%} {%- set _original_lang = "und" -%} {%- if _meta -%} {%- if _meta.titles and _meta.titles.en -%}{%- set _book_title = _meta.titles.en -%} {%- elif _meta.title -%}{%- set _book_title = _meta.title -%}{%- endif -%} {%- if _meta.originalLang -%}{%- set _original_lang = _meta.originalLang -%}{%- endif -%} {%- endif -%} {%- set _chapter_data = load_data(path="data/library/" ~ _book ~ "/chapter-" ~ _chapter ~ ".json", required=false) -%} {%- set _anchor = "#c" ~ _chapter ~ "p" ~ _verse -%} {%- set _book_url = get_url(path="library/" ~ _book, lang=lang, trailing_slash=true) -%} {%- set _verse_ref = _book_title ~ " " ~ _chapter ~ ":" ~ _verse -%} {%- if _to_verse != _verse -%}{%- set _verse_ref = _verse_ref ~ "-" ~ _to_verse -%}{%- endif -%} {# RTL scripts get an explicit `dir` so the original line lays out correctly even inside an LTR document. Extend this list when we add new RTL source languages (currently Hebrew and Arabic). #} {%- set _is_rtl = _original_lang == "he" or _original_lang == "ar" -%}
